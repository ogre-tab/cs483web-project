# cs483web-project
Group project for CS483 Web Data

Requires Python 3

Required Modules:
* PyQt5
* whoosh
* sqlite3
* requests

To run:\
Clone the repository.

Navigate to the 'scraping' directory.\
Run 'scrape.py' followed by 'build_db.py'.

Navigate to the root directory.\
Run 'app.py' to start the flask webserver.
